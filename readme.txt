English Dictionary Commandline Application

Developed via Python 3.9.0

==========================================
Run the application.

Application will ask you to enter a word "Enter a word: "

Enter a word and it will show you the definition of that word, if there are multiple definitions it will list them.

If the word you entered is not in the dictionary, the application will give you a message and if there is, a possible action.

To close the application simply enter "/end'" when it asks a word to you.
